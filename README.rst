clickstack plugin for `Tutor <https://docs.tutor.edly.io>`__
############################################################

clickstack plugin for Tutor


Installation
************

.. code-block:: bash

    pip install git+https://github.com/bmtcril/tutor-contrib-clickstack

Usage
*****

.. code-block:: bash

    tutor plugins enable clickstack
